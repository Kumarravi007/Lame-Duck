# How To Install
Download the [Latest release](https://github.com/pierpaolo-canini/Lame-Duck/releases/latest) and unzip the plugin files in the C:\ProgramData\Autodesk\Revit\Addins\XXXX\ folder where XXXX is the Revit Version you're interested in

Tested in Revit 2019, 2020
